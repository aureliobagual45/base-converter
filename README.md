BASE CONVERTER

This was my first code, so the structure is not very optimized and there is a lot of repetition in the functions.

The main goal of this project was to understand the logic behind base conversion and practice low-level problem solving.

FEATURES

Convert between binary, octal, decimal and hexadecimal
Manual parsing of input strings
Manual implementation of base conversion logic
Input validation for each base

HOW IT WORKS

The program first converts any input base to decimal, and then converts from decimal to the target base.

All operations are done using basic C concepts like arrays, loops and arithmetic operations.

WHAT I LEARNED

How number base systems work internally
String manipulation using char arrays
ASCII-based digit handling
Building logic step by step without relying on libraries
